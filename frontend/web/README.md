# Department Engagement & Career Platform (DECP)

A modern, premium, and minimal web application for university departments to connect students, alumni, and faculty.

## Features
- **Feed**: Social feed for department updates, research findings, and announcements.
- **Jobs**: Career board for internships, part-time, and full-time opportunities.
- **Events**: Department calendar and RSVP system.
- **Research**: Collaboration hub for ongoing research projects.
- **Messages**: Real-time direct and group messaging.
- **Analytics**: Executive dashboard for department engagement metrics.
- **Admin**: Moderation and user management tools.

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS v4
- Zustand (State Management)
- Axios (API Client)
- Lucide React (Icons)
- Recharts (Analytics)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Copy `.env.example` to `.env.local` and configure your API base URL if needed.

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design System
The application uses a custom "Avant-Garde" design system built on top of Tailwind CSS, featuring:
- Near-black graphite backgrounds
- Charcoal / deep slate surfaces
- Soft white primary text
- Electric blue-violet accents
- Thin borders and soft shadows
- Large whitespace zones for a premium feel
